# Sales & Profit Analysis Dashboard using Power BI

## Project Overview
This project presents an interactive Power BI dashboard developed using the Superstore dataset. The dashboard provides insights into sales performance, profitability, and customer behavior across different regions, categories, and time periods.

## Tools Used
- Microsoft Power BI
- Superstore Dataset

## Dashboard Pages
### 1. Business Performance Overview
- Total Sales KPI
- Monthly Sales Trend
- Sales by Category
- Regional Sales Distribution
- Interactive slicers (Year, Category, City)

### 2. Sales & Profit Insights
- Total Profit and Quantity KPIs
- Sales vs Profit Trend
- Profit by Category
- Segment-wise Sales Distribution
- Detailed Sales Summary Table
